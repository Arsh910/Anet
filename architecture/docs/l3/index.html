Level 3 · Index

The core, file by file.

Level 1 gave you the mental model. Level 2 opened the engine's hood. Level 3 is the line-by-line walkthrough of every file in anet/core/, in the order you should read them. Each page is a guided tour: the actual source on the left intent, the reason for every choice on the right.

How to read Level 3

Files are ordered top-down by dependency: we start with the entry point (engine.py, the file that orchestrates everything else), and each time it imports something we haven't seen, that file becomes the next chapter. By the time you reach paths.py at the bottom, every line in the core makes sense in context.

Each file gets its own folder with one overview page (N_filename.html) and sub-pages (Na_…, Nb_…) when the file is too long for one screen. The reading order is given by the leading number, the letter sub-orders within a folder.
